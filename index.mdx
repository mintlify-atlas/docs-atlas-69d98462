  <h2 className="text-3xl font-bold text-gray-900 dark:text-gray-100 mb-4">Quick Start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">Get Proton running in seconds</p>
  
  <Steps>
    <Step title="Install Proton">
      Install Proton with a single command:
      
      ```bash
      curl https://install.timeplus.com/oss | sh
      ```
      
      Or use Docker:
      
      ```bash
      docker run -d -p 8123:8123 -p 8463:8463 --name proton \
        d.timeplus.com/timeplus-io/proton:latest
      ```
    </Step>
    
    <Step title="Start the Server">
      Launch Proton server and client:
      
      ```bash
      proton server  # In one terminal
      proton client  # In another terminal
      ```
      
      <Info>On Mac, you can also use Homebrew: `brew install timeplus-io/timeplus/proton`</Info>
    </Step>
    
    <Step title="Run Your First Query">
      Create a stream and query real-time data:
      
      ```sql
      -- Create a stream with random data
      CREATE RANDOM STREAM devices(
        device string default 'device'||to_string(rand()%4),
        temperature float default rand()%1000/10);
      
      -- Query streaming data
      SELECT device, count(*), avg(temperature)
      FROM devices 
      GROUP BY device;
      ```
      
      <Accordion title="See example output">
      ```
      ┌─device──┬─count()─┬─avg(temperature)─┐
      │ device0 │    2256 │             49.8 │
      │ device1 │    2260 │             50.2 │
      │ device3 │    2259 │             49.5 │
      │ device2 │    2225 │             50.1 │
      └─────────┴─────────┴──────────────────┘
      ```
      </Accordion>
    </Step>
  </Steps>
</div>
